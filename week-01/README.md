# Цель недели

Научиться писать и запускать простой JS-код в Node.js, понимать базовые конструкции и не бояться ошибок.

## Задание A — базовые функции (functions.js)

```
function sum(a, b) {
  // ...
}

function isEven(n) {
  // ...
}

function max(arr) {
  // arr — массив чисел
}
```

### Требования

- Использовать let / const
- Без console.log внутри функций
- Функции возвращают значения
- max([]) должен вернуть null

## Задание B — простое число (prime.js)

Напиши программу, которая:

- принимает число из аргументов командной строки
- выводит в консоль:
- true — если число простое
- false — если нет

Требования

- Использовать цикл
- Обработать неверный ввод (node prime.js abc)

### Запрещено (важно)

❌ var

❌ браузерный JS

❌ копипаста из готовых решений

❌ всё в одном файле

## Start program

node index.js [number]
